## React Pokedex

### Requirements

- Node v9.3.0 or greater

### Setup

- Checkout repo
- `yarn add` to install dependencies
- `yarn start` to run
- Catch em all!

#### Note

When running localhost, cors must be allowed, i use the `Allow-Control-Allow-Origin: *` for chrome.
